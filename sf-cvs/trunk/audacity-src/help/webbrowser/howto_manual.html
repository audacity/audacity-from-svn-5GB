How to use this manual

In this manual you'll find lots of common screen elements, that you can rely on to navigate this manual.

TOP LEFT CORNER -> Back to the main page
TOP MIDDLE -> Back to the previous page, in this case the Welcome page
TITLE GRAPHICS -> Back to the previous page, in this case the Welcome page
TOP MIDDLE,one down -> Links to stuff in the same category.
All the links on this page will take you to this page again.
BOTTOM MIDDLE -> Back to the previous page, in this case the Welcome page
Not on every page.
BOTTOM LEFT CORNER -> Back to the main page

To reiterate, to get back to the main contents page, check the top-left corner of the page. To return to the page from which you came, check the top-middle of the page. Any title graphics are linked to the previous page too. Any stuff that fits in to the same category of the current page is usualy available as a link just above the page title and below the link back to the previous page. That way you can jump around all the tutorial pages that belong together for example.
