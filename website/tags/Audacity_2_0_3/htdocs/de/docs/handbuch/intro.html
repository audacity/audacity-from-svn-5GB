Audacity ...

Audacity ist ein kostenloser, betriebssystemunabhängiger Audio Editor.

Benutzer die Audacity das erste Mal benutzen, sollten zuerst die Einführung in 5 Schritten lesen, um grundlegende Informationen in Bezug auf das Digitales Audioformat, die Optionen in Audacity, als auch TonWiedergabe und die erste Aufnahme zu erhalten.

Was kann ich mit Audacity machen ?

Audacity ist ein Programm das digitale "Audio Waveforms" (Anm.d.Übers.: auf gut deutsch Lieder, Sprache usw.) bearbeitet. Es kann nicht nur direkt mit dem Programm aufgenommene Streams bearbeiten, sondern auch verschiedene Formate, wie WAV, AIFF, MP3, and Ogg Vorbis

Ebenso kannst du alle herkömmlichen Dateioperationen, wie

  • Schneiden, Kopieren, Einfügen, Löschen, Stille, Duplizieren, Teilen
  • Effekte zu jedem Ton hinzufügen
  • ein eingebauter Lautstärkeregler
  • veränderbarer Spektrogrammanzeige
  • Frequenzanalyse für die Audioanalyse>
  • für jedes Lied eine eigene Projektfrequenz
  • einfache und komplizierte Änderungen für einzelne Lieder (oder Gruppen)

Was kann Audacity nicht?

Obwohl Audacity ein sehr guter Audioeditor ist, mit dem man eine unbegrenzte Anzahl an Audiospuren verwalten kann, die auch noch beliebig gross sein können, kann Audacity nicht alles. Es kann nicht :

  • Mehr als 2 Kanäle gleichzeitig aufnehmen. Ebenfalls unterstützt es nicht bestimmte Hardware, sondfern arbeitet mit der im Rechner vorhandenen normalen Hardware zusammen.
  • Audacity öffnet Midi dateien, aber es ist kein Midi - Editor und die Möglichkeiten Midi zu bearbeiten, sind sehr beschränkt.

Wie arbeitet Audacity - ein Schnelleinstieg für erfahrene Benutzer

Audacity arbeitet mit Audiotracks, die eine Audiodatei enthalten. Diese Datei ist änderbar und alle Tätigkeiten können rückgängig gemacht werden.

Nahezu alles kann rückgängig gemacht werden, inklusive Löschen von Dateiteilen und einfügen anderer Dateien. Dieses geschieht blitzschnell.

Es gibt eine Master Gain Kontrolle in der "Toolbar"

Die Windows and Mac Versionen von Audacity können VST plugins benutzen, bisher gibt es jedoch noch keine Vorschau Funktion.

Die automatische Lautstärkenkontrolle über das envelope tool.

Änderungsfunktionen sind, Schneiden, Kopieren und Einfügen, Verdoppeln (in eine neue Datei ), Aufteilen (ebenfalls in eine neue Datei ), Stille und Einfügen Stille.

Was macht Audacity einzigartig?

  • Audacity ist kostenlos, die Programmcode ist freiverfügbar unter der GNU General Public License
  • Audacity ist betriebssystemunabhöngig - es läuft unter Windows (98 bis XP ), MAC OS 9, MAC OS X, und verschiedenen Unix Platformenen, ebenfalls LINUX
  • Keine Beschränkung in der Anzahl und Grösse der Dateien mit Ausnahme deiner Festplatte
  • Der Import von Dateien umfasst nahezu alles: WAV,AIFF, Next/AU, IRCAM, MP3 und Midi werden direkt unterstützt, aber Audacity öffnet nahezu jeden nicht reduzierten Audiodatenstrom und erkennt das Format (siehe auch Import Raw Data.. )
  • Audacity beinhaltet nicht nur einige, bereits fest eingebaute Effekte, sondern unterstützt Plugin's die auf dem Industriestandard VST basieren. Es gibt dutzende von kostenlosen oder auch Shareware - Effekten die so eingebunden werden können, von Reverb bis Noise reduction.
  • Audacity arbeitet als normaler Editor, ohne die Dateien zu zerstören oder zu ändern , mit einer fast unbegrenzten Anzahl an Undo's und schreibt dabei die Änderungen an der Datei auf die Festplatte, so daß keine Notwendigkeit besteht das ganze in "Real-Time" zu berechnen.
  • Label Dateien geben dir die Möglichkeit, Wellenformen zu kommentieren und zu bearbeiten (zum Beispiel Die Übersetzung von Sprache) und das ganze später in eine Textdatei zu exportieren.
  • Die Spektralanalyse erlaubt die die Dateien (Waveforms) in Spektrogrammen anzuschauen, die zu verschieben, egal wohin innerhalb der Audiodatei und ebenso das ganze zu exportieren.
