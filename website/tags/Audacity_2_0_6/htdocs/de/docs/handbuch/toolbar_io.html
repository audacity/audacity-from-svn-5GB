I/O Sliders and Input Selector

Output Input Input Selector

These sliders control the mixer settings of the soundcard in your system.

The sliders will jump to zero(left most position) if the drivers do not expose these functions. This is the case with some digital I/O cards of RME and possibly others as well. On these cards you will not be able to control the anything with these sliders.

Output Slider

This is the left hand slider that lets you control the output level of your soundcard. It actualy controls the output setting of the soundcard driver.

Should your output sound distorted, this is not the first place to look. You might get lucky though, so give it a quick slide. Usualy however, the distortion comes from the mix of your project clipping. You actualy need to reduce the level of your tracks. It is suggested to reduce every track by the same amount.

Input Slider

This is the right hand slider that lets you control the input level of the input type selected in the Input Slider Selector. It actualy controls the that particular setting of the soundcard driver.

What you record is also determined in the mixer software of your soundcard. There you can switch certain sources on and off, such as the Line Input, the Microphone Input or the CD Audio.

After selecting the corresponding input source in the Input Slider Selector, you can control the level of that source going in to the recording with this slider.

Should your input sound distorted, you should lower this slider untill no distortion can heard.

For recording, it is advised to turn down all sources you do not wish to record. If turned up they only add noise.

Input Selector for the Input Slider

Pick the input source you wish to control with the Input Slider. All these items are exposed by the soundcard driver, so this will vary with different soundcards.
