Créer des raccourcis clavier

Où ?

Dans la boîte de dialogue préférences, allez à la section  Clavier ou éditez le fichier de configuration clavier directement avec un éditeur de texte. Vous le trouverez dans le dossier où vous avez installé Audacity sous le nom de Audacity-keys.xml .

Audacity n'exécute pas cette configuration au démarrage, mais la sauve dans la base des registres(Windows). Si vous voulez modifier le fichier  Audacity.xml  avec un éditeur de texte, vous devrez ensuite charger ce fichier avec le bouton "Charger" pour qu'Audacity l'utilise. mais vous pouvez aussi sauvegarder sous un autre nom pourvu qu'il ait une extension XML et contienne les différents tableaux de clefs décrites ci-après.

Audacity ne stockera pas le tableau courant dans Audacity.xml . Pour sauvegarder vos changements dans le tableau courant, sauvegardez avec le bouton "Enregistrer" dans un fichier XML. Après l'avoir édité,  vous devrez le charger à nouveau, pour qu'il soit pris en compte.

Editer et Créer des raccourcis avec la boîte de dialogue Préférences

Changer un raccourci :

  1. Cliquez sur la commande dans la liste. Elle passe en surbrillance

  2. Cliquez sur le bouton modifier. Si un raccourci était assigné à cette commande, il sera affiché dans cette boîte.

  3. Saisissez la clé ou combinaison de clés souhaitées.
  4. Cliquez ensuite sur le bouton Enregistrer pour sauvegarder votre modification.

Effacer un raccourci :

  1. Cliquez sur la commande dans la liste. Elle passe en surbrillance.
  2. Cliquez sur le bouton effacer.

Editer et Créer des raccourcis avec un éditeur de texte

Les raccourcis clavier que vous avez édité avec la boîte de dialogue sont stockés dans un fichier XML. Aussi, vous pouvez l'éditer avec un éditeur de texte.

Chaque tableau de clefs commence par

<audacitykeyboard audacityversion="1.2">

et finit avec

</audacitykeyboard>

Chaque commands est définie avec une étiquette de commande . Un  exemple :

   <command nom="Annuler" label="Annuler" Clef="Ctrl+Z" />.

nom

La commande actuelle qu'Audacity connaît.

label

Description  de la commande qui a ét vu dans la section Editeur de cartes de Préférences .

clef

C'est le raccourci actuel. Contrôler quel est le format dans la boîte de dialogue des Préférences.
